0.1.0
-----

Initial release.

- :func:`~schlepper.deploy` for deploying static assets to Cloudflare Pages.
- :class:`~schlepper.ApiToken` and :class:`~schlepper.ApiKey` credential types.
- Support for special files (``_headers``, ``_redirects``, ``_routes.json``).
- Concurrent asset uploading with automatic retries.
